Windows PHPUnit

  1. Download Composer for windows using the installer. Found here.
  2. Follow the steps in the installer.
  3. Follow the instructions for installing phpunit using composer. Found here. Ensure you have Git Bash installed.
  4. Run the following command: "./vendor/bin/phpunit tests"
  5. Add any more test files to the tests directory. Insure the files end in [anyname]Test.php.
